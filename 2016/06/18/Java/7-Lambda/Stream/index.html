Stream 流

先看看Java里面是怎么定义Stream的：

A sequence of elements supporting sequential and parallel aggregate operations.

上面的那句话意思是：

  • Stream是元素的集合，这点让Stream看起来用些类似Iterator；
  • 可以支持顺序和并行的对原Stream进行汇聚的操作；
